# PhotoFinder - Organizador Inteligente de Fotos

**Este é o projeto PhotoFinder, NÃO é o aerocost.**

## Identificação do Projeto

- **Nome:** PhotoFinder
- **Descrição:** Organizador Inteligente de Fotos com IA + Google Drive
- **Repositório:** https://github.com/robsonpaulista/appfotosjadyel
- **Frontend:** Next.js (diretório `frontend/`)

## Configuração Vercel

- **Root Directory:** `frontend`
- **Framework:** Next.js
- **Build Command:** `npm run build`
- **Output Directory:** `.next`

## Estrutura

```
/
├── frontend/          # Aplicação Next.js (PhotoFinder)
├── backend/           # Backend Express (não usado no Vercel)
├── database/          # Schemas SQL
└── vercel.json        # Configuração Vercel
```

**IMPORTANTE:** O Vercel deve usar o diretório `frontend/` como root.
