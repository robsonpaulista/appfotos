# PhotoFinder - Deploy Vercel

Este é o projeto **PhotoFinder**, não aerocost.

## Configuração Vercel

- **Root Directory:** `frontend`
- **Framework:** Next.js
- **Build Command:** `npm run build`
- **Output Directory:** `.next`

## Verificação

Se você está vendo o app "aerocost", verifique:

1. O repositório conectado no Vercel está correto: `robsonpaulista/appfotosjadyel`
2. O Root Directory está configurado como `frontend`
3. Não há outro projeto do aerocost conectado ao mesmo domínio
